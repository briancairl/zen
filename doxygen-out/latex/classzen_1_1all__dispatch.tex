\subsubsection*{template$<$typename... Invocable\+Ts$>$\newline
class zen\+::all\+\_\+dispatch$<$ Invocable\+Ts $>$}

Implements invocable dispatch behavior for free-\/function {\ttfamily all} 

Executes invocables until any of them returns an invalid result, then terminates, returning a result holding the first invalid status. Otherwise, returns a result containing values returned by all invocables. ~\newline
This is the default, single threaded implementation. Specializations of \mbox{\hyperlink{classzen_1_1all__dispatch}{all\+\_\+dispatch}} may be made available for invocation in different execution contexts, such as multi-\/threaded dispatch.
