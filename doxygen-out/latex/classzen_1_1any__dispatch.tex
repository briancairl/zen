\subsubsection*{template$<$typename... Invocable\+Ts$>$\newline
class zen\+::any\+\_\+dispatch$<$ Invocable\+Ts $>$}

Implements invocable dispatch behavior for free-\/function {\ttfamily any} 

Executes invocables until any of them returns a valid result, then terminates, returning that result. Otherwise, returns result with the last invalid status. ~\newline
This is the default, single threaded implementation. Specializations of \mbox{\hyperlink{classzen_1_1any__dispatch}{any\+\_\+dispatch}} may be made available for invocation in different execution contexts, such as multi-\/threaded dispatch.
