Detailed Description

template<typename T>
class zen::result_value< T >

Holds a value whose constructor/destructor can be manually invoked.

This is used to deal with objects which have non-default constructors:

  • Defers construction of a value of type T to manual call to emplace
  • Requires manual destruction with destroy
